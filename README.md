# Freya E-Commerce - React / Firebase 👥

Este es un proyecto Ecommerce moderno desarrollado con React y Vite, integrado con Firebase para la gestión de datos y autenticación. La aplicación está diseñada para ser rápida, escalable y ofrecer una experiencia de usuario fluida mediante el uso de rutas dinámicas.

## Características 💬 <a id="caracteristicas"></a>

- **Interfaz optimizada:** Desarrollada con React y Vite para garantizar tiempos de respuesta rápidos y un rendimiento fluido.

- **Navegación dinámica:** Implementada con React Router DOM para una experiencia fluida entre páginas.

- **Backend con Firebase:** Gestión de persistencia de datos mediante Cloud Firestore y manejo de usuarios con Firebase Authentication.

- **Estado Global (Context API):** Uso de AuthProvider para la sesión del usuario y CartProvider para el manejo del carrito de compras en toda la aplicación.

- **Rutas Protegidas:** Implementación de seguridad en el frontend para restringir el acceso a la sección de "Agregar productos" únicamente a usuarios autorizados.

- **Modo Oscuro (Dark Mode):** Funcionalidad de tema visual con persistencia automática mediante localStorage.

- **Arquitectura Modular:** Código organizado en componentes, páginas y contextos, facilitando el mantenimiento y la escalabilidad.

## Tecnologías 🌐 <a id="tecnologias"></a>

- **Vite**: Entorno de desarrollo y empaquetado.

- **React**: Biblioteca principal para la interfaz de usuario.

- **Firebase (Auth & Firestore)**: Infraestructura de backend para datos y autenticación.

- **React Router DOM**: Sistema de gestión de rutas y navegación dinámica.

- **LocalStorage**: Utilizado para la persistencia del tema (Modo Oscuro).
